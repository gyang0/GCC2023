Program Information

Overview

This program simulates flooding, a consequence of rising sea levels. Click a point on the globe to see the potential way a flood could impact that area.

This doesn't "solve" climate change, but it does help mitigate short-term concerns. By focusing on the areas/routes more susceptible to flooding, resources can be saved. The saved resources can then be contributed to other efforts, like tackling climate change.

(Only a demo. Wildly inaccurate, due to program & data limitations.)

How to Use

Click any region on the globe to get the elevation and likely path of flood.

  • Elevation: approximate elevation of selected point on map
  • Lowest sea level: lowest sea level such that the selected area can still flood. In most cases it'll be close to the actual elevation, but it'll be higher if the area is surrounded by mountains or something.
  • Path of the likely flood path is marked in red.

  • The second box contains information about the barriers that can be constructed.
  • Barrier for blocking at coastline: Height of barrier required to block the flood if we're construting one at the coastline. Assuming that the flood follows the flood path.
  • Lowest barrier along flood path: Minimum height of a barrier required to block the flood if we can construct one at any point on the flood path.

Code Notes

  • The grayscale map I used for the elevation data was "scaled 0-6400m." I'm interpreting that as black - 0m, white - 6400m, which seems reasonable.

  • Map was converted from image to 2D array using a quick program in Java (coded within duration of Hackathon): https://github.com/gyang0/ImageConverter

  • The "globe" is a square image warped using circle equations.

  • How the path of likely flooding is determined:
    Binary search on all possible scaled sea levels 0-255 to find the lowest sea level where flooding is still possible (using BFS starting from selected point).
    Do BFS one more time to get the shortest path.
    Use the shortest path found from the BFS to mark squares on that path.
